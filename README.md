# [kanyilmaz.me](kanyilmaz.me)

## Run
```
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```

## RSS Feed

- Feed URL: `https://kanyilmaz.me/feed.xml`
- Generated by `jekyll-feed` (GitHub Pages-supported plugin).
- Posts are pulled from `site.posts` and emitted as Atom feed entries.

## Test RSS Locally

1. Build locally:
   ```bash
   LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8 bundle exec jekyll serve
   ```
2. Open the feed directly:
   - `http://127.0.0.1:4000/feed.xml`
3. Validate autodiscovery from pages:
   - `http://127.0.0.1:4000/`
   - `http://127.0.0.1:4000/blog/`
   - View page source and confirm:
     ```html
     <link rel="alternate" type="application/atom+xml" ... href="http://127.0.0.1:4000/feed.xml">
     ```
4. Confirm feed entries:
   ```bash
   curl -s http://127.0.0.1:4000/feed.xml | rg "<entry>|<title type=\"html\">"
   ```
5. Reader test (real-world):
   - Paste `http://127.0.0.1:4000/feed.xml` into a local-compatible reader, or
   - After deploy, paste `https://kanyilmaz.me` into Feedly/Inoreader and confirm auto-detection.
